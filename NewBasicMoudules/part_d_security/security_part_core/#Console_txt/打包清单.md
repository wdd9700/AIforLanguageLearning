## ? 快速启动步骤

### 1. 解压文件

保持文件夹结构：
```
security_part_core/
├── auth_core.py
├── auth_routes.py
├── ...
```

### 2. 安装依赖

```bash
cd security_part_core
pip install -r requirements.txt
```

### 3. 配置环境变量

```bash
# Windows PowerShell
$env:JWT_SECRET_KEY="your-32-char-random-secret-key-here"

# Linux/Mac
export JWT_SECRET_KEY="your-32-char-random-secret-key-here"
```

### 4. 启动服务

```bash
python main.py
```

服务启动后访问：http://localhost:8001
