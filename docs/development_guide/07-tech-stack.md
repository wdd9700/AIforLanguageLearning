## 八、技术选型理由

### 8.1 为什么选择 FastAPI?

- 原生异步支持，性能优异
- 自动 API 文档生成
- Pydantic 集成，类型安全
- Python 生态丰富

### 8.2 为什么选择 Vue 3?

- 组合式 API，逻辑复用方便
- TypeScript 支持好
- 性能优秀，体积小
- 中文文档完善

### 8.3 为什么选择边云协同?

- 降低 API 调用成本
- 保障实时性要求
- 保护用户隐私
- 充分利用本地硬件
